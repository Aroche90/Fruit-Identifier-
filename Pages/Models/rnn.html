Recurrent Neural Networks:

A class of artificial neural networks where connections between nodes form a directed graph along a temporal sequence.

I used a recurrent neural network (RNN) for image classification. The RNN the current input example they see, and also input that has been used previously to a certain extent. If all the hidden layers have the same weights and bias then you can merge them together and each step will go through the hidden layer and a recurrent neuron stores the information from each stepped that has gone through and merges it with the current step. RNN is optimized for predicting what is going to come next, particularly for language, such as which letter or word is going to come next.
